# Image-mixer
## Description
it application used for the fourier transform  image represnetation and to mix between to image by both 
- magnitude and phase 
- real and imaginary
## Features
- used to represent the magnitude-phase 2d image 
- mix between two image

- PIL
- logging
- cv2
